# CheSSv3

A simple chess game.

## Description
CheSSv3 is a simple terminal-based chess game, initially developed in 2003 (v1), continued in 2008, and partially rewritten/almost completed in 2020 (v3).

## Getting Started

### Dependencies
None, only standard C library

### Installing

$ make

### Executing program

$ ./chess
